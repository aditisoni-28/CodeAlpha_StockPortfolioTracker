# CodeAlpha_StockPortfolioTracker

## Project Overview

Stock Portfolio Tracker is a Python application that allows users to create a stock portfolio, calculate investment values, and generate portfolio summaries using predefined stock prices.

## Features

- Multiple stock selection
- Quantity-based investment calculation
- Portfolio summary report
- Portfolio allocation percentage
- Input validation
- File handling for report generation
- User-friendly interface

## Technologies Used

- Python 3

## How to Run

1. Clone or download the repository.
2. Open a terminal in the project folder.
3. Run:

```bash
python stock_portfolio_tracker.py
```
